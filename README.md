# Saves-PWD

with this project you may save your secret keys safely. All of your secret keys are saved encrypted in database

### this project uses
- backend
  - FastApi
  - mongodb
- frontend
  - Vuejs
  - vuetify
- docker/docker-compose (orchestration)

### to run this project:
- have docker
- have docker-compose 
- rename .env.example to .env

### Rode os comandos:
```bash
$ docker-compose build
$ docker-compose up
```
